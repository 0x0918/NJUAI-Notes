$\because f(x)=x^3-3x+t$

$\therefore f'(x)=3x^2-3$

            3       0       -3
    --------------------------
    1       0       -3      t
    1       0       -1
    -------------------------
                    -2      t

            -2      t
    ------------------
    3       0       3
    3       -3t/2
    ------------------
            3t/2    3
            3t/2    -3t^2/4
            ---------------
                    0

$\therefore 3=-\displaystyle\frac{3t^2}{4}$

$\therefore t^2=-4$

$\therefore t=\pm2i$
