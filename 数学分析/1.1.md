## 集合论

### 集合相等

要说集合$A$和集合$B$相等,   
即是说明 $A \subset B \ \& \ B \subset A$

### 交集运算律

交换律: $A \cap B = B \cap A$

结合律: $A \cap B \cap C = A \cap (B \cap C)$

诸如此类...

### 笛卡尔集合

$A \times B = \{ (x, y) ; x \in A, y \in B \}$

## 无限集合

* 可数集合 (可按照一定规则列出来的集合)
  * 自然数$\mathbb Q$
* 不可数集合 (不可按照一定规则列出来的结合)

### 无限可数集合
$A = B = \mathbb Q$  
$C = A \times B = \{ (x, y) ; x \in A, y \in B \}$  

则$C$也是一个无限可数集合
