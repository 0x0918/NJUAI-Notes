## 3.

### T2: 空元素

| $X$ | $X+1$ | $X\cdot 0$ |
| --- | ----- | ---------- |
| 0   | 1     | 0          |
| 1   | 1     | 0          |

$\therefore X+1=1, X\cdot 0=0$

### T3: 同一律

| $X$ | $X+X$ | $X\cdot X$ |
| --- | ----- | ---------- |
| 0   | 0     | 0          |
| 1   | 1     | 1          |

$\therefore X+X=X, X\cdot X=X$

### T4: 还原律

| $X$ | $\overline{X}$ | $\overline{\overline{X}}$ |
| --- | -------------- | ------------------------- |
| 0   | 1              | 0                         |
| 1   | 0              | 1                         |

$\therefore \overline{\overline{X}}=X$

### T5: 互补律

| $X$ | $X+\overline{X}$ | $X\cdot \overline{X}$ |
| --- | ---------------- | --------------------- |
| 0   | 1                | 0                     |
| 1   | 1                | 0                     |

$\therefore X+\overline{X}=1, X\cdot \overline{X}=0$
