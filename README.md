# Notes

我在NJU人工智能学院上课的笔记.

## 如何查看?

下载安装Vscode, git, 安装`Markdown Preview Enhance`插件.  

使用命令

```bash
git clone https://github.com/OrangeX4/Notes.git --depth=1
```

因为内含图片, 略大, 耗时长, 请耐心等待.

Clone完成之后用Vscode打开该文件夹.

点开一个你想看的markdown文件, 按下快捷键`Ctrl + K`, 再按下`V`键即可.
