## 如何查看?

下载安装 VSCode, git, 并安装 `Markdown Preview Enhance` 插件.  

使用命令:

```bash
git clone https://github.com/OrangeX4/Notes.git --depth=1
```

下载这个 Repo.

因为内含图片, 略大, 耗时长, 请耐心等待.

Clone 完成之后用 VSCode 打开该文件夹.

点开一个你想看的 Markdown 文件, 按下快捷键 `Ctrl + K` , 再按下 `V` 键即可.
