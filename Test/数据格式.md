# 接口文档

## ESG 数据获取

接口：

```http
http://122.9.148.74:7685/esg?name=<名称>
```

例如:

```http
http://122.9.148.74:7685/esg?name=万马股
```

会获得返回：

```json
{
    "status": "success",
    "data": {
        "e_score": 4.955555555555556,
        "s_score": 4.644444444444444,
        "g_score": 3.0833333333333335
    }
}
```

失败时返回：

```json
{
    "status": "failure",
    "cause": "no such company."
}
```

## ESG 股票推荐

通过对 ESG 分数排序进行推荐.

例如:

```http
http://localhost:7685/esg_recommended?by=e_score&num=5
```

其中 `by` 取值 `e_score`, `s_score`, `g_score`. 而 `num` 代表了获取的数量.
